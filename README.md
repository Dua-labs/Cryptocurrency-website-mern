## Cryptocurrency Website Development Using MERN Stack

## 📖 Project Overview
This is a full-stack Cryptocurrency Website developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The project provides secure user authentication, cryptocurrency market data, portfolio management, watchlist, transaction history, and an admin dashboard.

## ✨ Features
- User Authentication (Login & Register)
- Cryptocurrency Market Data
- Portfolio Management
- Watchlist
- Transaction History
- Real-time Data
- Admin Dashboard
- Responsive User Interface

## 🛠️ Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js
- JWT Authentication
- Socket.io
- CSS
- Vite

## 🚀 Installation

### Clone Repository
bash
git clone https://github.com/Dua-labs/Cryptocurrency-website-mern.git


### Install Backend
bash
cd backend
npm install
npm run dev


### Install Frontend
bash
cd frontend
npm install
npm run dev
